# 市场规模测算原则

---

## 核心原则：双重验证

**任何市场规模结论，必须用需求导向和供给导向两种方法分别测算，相互印证。**

单一方法得出的数字，默认证据不足。两种方法得出的区间如果差距超过 50%，需要找出原因，不能取平均值蒙混过关。

---

## 三种测算方法

### 需求导向法（测 TAM 潜在市场）

```
基础公式：
TAM = 目标客户数 × 渗透率 × 客均订单量 × 客单价

含存量替换：
TAM = 目标客户数 × [新增购买渗透比例 + (存量渗透比例 × 1/使用寿命年限)] × 客单价
```

**适用场景**：消费品、To C 服务、有明确用户基础的行业

**关键变量**：渗透率是最重要的假设，需引用历史趋势或海外对标数据支撑，不能拍脑袋设定

### 供给导向法（测 SOM 可获得市场）

```
SOM = 最小运作单位产能上限（受最慢瓶颈环节约束）× 产能利用率 × 客单价

步骤：
1. 拆解交易流程，找最慢环节（瓶颈）
2. 计算瓶颈环节单位时间产能上限
3. 乘以时间窗口和利用率
```

**适用场景**：制造业、To B 服务、产能约束型行业

### 供需匹配法（稳定比例反推）

```
总量 = 已知参照物数量 × 稳定比例系数

例：咖啡豆市场 = 咖啡杯销量 × 15克/杯
例：金融服务 = 服务人口 ÷ 77人/从业者
例：车桩配比 = 电动车保有量 × 政策目标车桩比
```

**适用场景**：中间品市场、辅助商品市场、配套市场

## 无公开数据时的另类测算路径

当行业处于"来不及统计"（新生业态）、"没动力统计"（细分太小）、"做不到统计"（极度不透明）时，放弃找现成数据，转为自行测算：

- **爬虫抓取**：从电商平台/社区 App 抓取实时销售数据和用户活跃度
- **草根调研**：门店蹲点统计客流、收集真实销售小票（浑水做空瑞幸的做法）
- **遥感卫星**：分析卫星图片预测产量（农作物、油罐存量等）
- **分层细化**：按城市层级分别测算再加总（禁止用全国总量粗暴乘法）
  - 示例：电梯保有量需按一线 50%/新一线 55%/三线 60% 分层计算
- **空间对标锁定天花板**：对标欧美日韩成熟市场稳态数据作为假设上限
  - 示例：中国人均用纸量对标日本 15-16kg/美国 22kg，设天花板为 20kg

## 常见测算错误

**❌ 粗放测算陷阱**
用全国总人口做粗暴乘法，未按城市层级分层测算。正确做法：按一线/新一线/三线及以下分别计算后加总。

**❌ 空间对标陷阱**
照搬海外模式，未验证政治/经济/文化/技术多维相似性。必须用 PEST 框架逐项检验相似度，不能默认"中国会走日本的路"。

**❌ 渗透率拍脑袋**
设定渗透率假设时没有历史趋势或海外数据支撑。渗透率是整个需求导向法最敏感的变量，需要独立论证。

**❌ 两种方法结果差异大但不说明**
需求导向测算 1000 亿，供给导向测算 500 亿，直接取均值 750 亿。正确做法：找出差异原因（定义不一致？渗透率假设？），解释后再决定用哪个数。

## 关键阈值参考

| 指标 | 阈值 | 含义 |
|------|------|------|
| 细分 TAM | < 500 亿元 | 不具备孕育独角兽的市场条件 |
| 渗透率 | 15%~20% | 导入期→成长期拐点 |
| 渗透率 | 35%~40% | 成长期→成熟期拐点 |
| CAPEX/折旧摊销 | ≤ 1.5 | 产能格局改善信号（买入参考） |
| CAPEX/折旧摊销 | < 1.0 | 行业绝对底部（强买入信号） |

---

## 产能周期 CAPEX 计算注意

```
CAPEX = 购建固定资产、无形资产和其他长期资产支付的现金
      - 处置固定资产、无形资产和其他长期资产收回的现金净额
```

注意工程尾款结算的时间滞后，不要误判扩产节奏。

---

## 一句话原则

双重验证不是形式主义——两种方法结论一致才能建立信心；结论差异才是需要追问的地方。
