# 报告输出结构

```
一、行业定义与边界（含产业链图）
二、生命周期定位（渗透率判断）
三、市场规模（需求导向+供给导向双验证）
四、增长驱动因素分析
五、竞争格局（护城河排查+产能周期）
六、价值链与利润池分布
七、景气度高频跟踪指标
八、核心风险
九、投资结论（周期位置+标的方向+时机判断）
```

## 汇报顺序（SCQR原则）

SCQR = Situation（现状）、Complication（矛盾/变化）、Question（核心问题）、Resolution（结论/答案）。

- 面向大老板/投委会：先给结论（R），再补逻辑（SCQ）
- 面向分析师/同事：标准叙事顺序（SCQR）
- 禁止“做饭思路”汇报（假设→分析→结论），受众会失去耐心

## 输出深度

- 快速判断版：保留核心结论、阶段判断、市场规模摘要、竞争格局摘要、投资判断
- 标准报告版：按完整章节展开
- 披露/底稿版：在标准报告版基础上补来源、时间、口径、假设与可复核链路
