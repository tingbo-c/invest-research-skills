# 行业数据来源速查

按数据类型和可信度排列。优先使用一手数据，第三方报告作为辅助验证。

---

## 数据来源优先级原则

行业研究里，不同来源的权重不能一样。结论应按下面顺序赋权：

| 优先级 | 数据类型 | 代表来源 | 用法 |
|------|---------|---------|------|
| P1 | 官方一手统计/监管披露 | 国家统计局、海关总署、央行、部委、监管机构 | 用于定义行业方向、规模基线、政策约束，是最主要依据 |
| P2 | 行业协会与原始经营披露 | 行业协会、上市公司年报/季报/招股书、公司公告 | 用于细分赛道拆解、产业链验证、竞争格局判断 |
| P3 | 可交叉验证的交易/业务数据 | 招投标、海关HS细项、价格库存、门店网点、渠道销售 | 用于验证景气、订单、价格、库存、项目落地 |
| P4 | 第三方研究与数据库 | 券商研报、咨询公司、商业数据库 | 用于补方法、补口径、补行业框架，不单独作为结论依据 |
| P5 | 另类与行为数据 | 舆情、招聘、搜索指数、地图热力、App活跃 | 只能用于验证变化方向或辅助找线索，不能单独证明行业规模、竞争格局或投资结论 |

**核心要求：**

- P1 和 P2 是主证据层，能决定结论主框架。
- P3 是验证层，适合捕捉拐点和修正滞后数据。
- P4 只能辅助，不能替代原始数据。
- P5 权威性最低，只能做旁证，不能单独上结论。

**特别提醒：**

- 舆情热度高，不等于需求真实。
- 招聘岗位多，不等于收入兑现。
- App活跃增长，不等于商业模式成立。
- 地图热力和门店客流回升，不等于利润改善。

这些数据最多说明“变化可能正在发生”，不能说明“结论已经成立”。

## 宏观/通用数据（自上而下估算基础）

| 数据类型 | 来源 | 说明 |
|---------|------|------|
| GDP、消费数据 | 国家统计局 stats.gov.cn | 最权威，但有滞后 |
| 行业产值/产量 | 各行业主管部委 | 工信部、商务部、农业农村部等 |
| 进出口数据 | 海关总署 customs.gov.cn | 判断行业对外依存度 |
| 人口/城镇化数据 | 国家统计局 | 消费品行业必查 |
| 上市公司财务数据 | Wind、Choice、同花顺 | 年报拆解最可信 |
| 价格/通胀数据 | 国家统计局、发改委价格监测中心 | 看PPI、CPI、工业品和居民端价格变化 |
| PMI/景气指标 | 国家统计局、中国物流与采购联合会 | 判断制造业和服务业景气方向 |
| 能源生产与消费 | 国家能源局、国家统计局 | 电力、煤炭、油气、新能源装机等基础数据 |
| 财政/税收数据 | 财政部、税务总局 | 看地方财政压力、税收结构、政策支持力度 |
| 就业/收入数据 | 国家统计局、人社部 | 验证消费能力、劳动力供给和工资趋势 |
| 利率/社融/汇率 | 中国人民银行、国家外汇管理局 | 看融资环境、信用扩张和外需传导 |
| 企业注册/注销 | 国家企业信用信息公示系统、企查查、天眼查 | 看新进入者、退出速度和行业活跃度 |
| 招投标/采购数据 | 中国政府采购网、全国公共资源交易平台 | 看订单落地、项目密度和客户结构 |
| 专利/技术数据 | 国家知识产权局、Google Patents | 看技术迭代、专利壁垒和研发方向 |
| 标准/认证数据 | 国家标准全文公开系统、行业标准公告 | 判断行业规范、准入门槛和替代节奏 |

## 专家访谈与专业社区（一手认知获取）

本节属于 P3（专家访谈有原始价值）或 P5（社区观点仅作参考）。

| 类型 | 来源 | 说明 |
|------|------|------|
| 专家访谈平台 | 凯盛融英、GLG、In On Advisory | 约见行业专家，适合获取产业链细节和定性判断，需注意专家立场 |
| 约见行家 | 在行 App、知识星球 | 适合细分领域的从业者访谈，成本较低 |
| 投资者社区 | 雪球、股吧 | 可追踪市场情绪和散户预期，注意区分专业判断与情绪噪音 |
| 微信生态搜索 | 搜狗搜索（微信文章专搜） | 适合搜索行业深度分析文章和自媒体研究，质量参差不齐 |
| 金融条件搜索 | i问财 | 支持按财务条件筛选上市公司，适合量化选股和行业横向比较 |

**使用原则：**
- 凯盛融英等专家访谈是 P3 级别，可用于验证产业链细节，但不能单独支撑规模测算结论
- 雪球、股吧等社区内容是 P5 级别，只能辅助判断市场情绪，不能作为基本面依据
- i问财筛选结果是 P2-P3，来源是上市公司原始财务数据，可信度较高

## 高频与另类数据（跟踪景气和验证拐点）

本节默认属于 P3-P5。

- 商品价格、库存、招投标、门店网点这类更接近 P3，可用于验证基本面变化。
- 招聘、舆情、搜索热度、地图热力、App 活跃这类更接近 P5，只能辅助判断方向。

| 数据类型 | 来源 | 说明 |
|---------|------|------|
| 商品价格/库存 | 生意社、百川盈孚、Mysteel、隆众资讯 | 大宗和中游材料景气度常用高频数据 |
| 航运/物流 | 上海航运交易所、交通运输部、中国物流与采购联合会 | 看运价、吞吐量、货运流向 |
| 电力/开工率 | 国家能源局、各地电网公告、卓创资讯 | 看工业活动强弱和产能利用率 |
| 地图热力/客流 | 高德、百度地图、美团、久谦中台等 | 看门店客流、商圈活跃度、线下修复 |
| 电商销售 | 淘宝天猫、京东、抖音电商、蝉妈妈、飞瓜 | 看SKU销售、价格带、促销节奏 |
| App活跃/流量 | QuestMobile、点点数据、七麦数据 | 互联网和平台型行业常用 |
| 招聘数据 | BOSS直聘、智联招聘、前程无忧、猎聘 | 看企业扩张节奏、岗位结构和薪资变化 |
| 舆情/搜索热度 | 百度指数、微信指数、巨量算数、Google Trends | 看用户兴趣和叙事热度，但不能替代基本面 |
| 卫星/遥感 | NASA、Sentinel、商业遥感服务商 | 用于农作物、矿山、港口、油罐等观测 |
| 门店/网点追踪 | 窄门餐眼、极海品牌监测、地图POI | 看开闭店、渠道扩张和区域渗透 |

## 上市公司年报拆解（自下而上估算核心）

**使用方法：**
1. 找到行业内A股/港股/美股主要上市公司
2. 从年报中找：营业收入、市占率（如有披露）、行业增速判断
3. 用"收入总和 ÷ 估计市占率"反推行业规模
4. 多家公司交叉验证，取合理区间

**注意：**
- 年报口径不统一时要调整（有的公司把上下游业务混在一起）
- 非上市公司市占率高的行业（如餐饮、农业），这个方法误差较大

## 使用顺序建议

1. 先用国家统计局、部委、海关、央行等官方数据定大盘和方向。
2. 再用行业协会、上市公司年报、招投标和海关细分口径做交叉验证。
3. 对没有公开统计的新行业，再补价格库存、电商、卫星、门店监测、地图热力、招聘、舆情等数据抓边际变化。
4. 第三方报告只用于补方法、补口径、补趋势，不单独作为结论依据。
5. 任何时候都不要让 P5 数据越级替代 P1-P3 数据。
