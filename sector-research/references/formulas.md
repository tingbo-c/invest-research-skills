# 测算公式与判断阈值速查

---

## 市场规模测算公式

### 需求导向法（TAM）
```
基础公式：
TAM = 目标客户数 × 渗透率 × 客均订单量 × 客单价

含存量替换：
TAM = 目标客户数 × [新增购买渗透比例 + (存量渗透比例 × 1/使用寿命年限)] × 客单价
```

### 供给导向法（SOM）
```
SOM = 最小运作单位产能上限（受最慢瓶颈环节约束）× 产能利用率 × 客单价

步骤：
1. 拆解交易流程，找最慢环节（瓶颈）
2. 计算瓶颈环节单位时间产能上限
3. 乘以时间窗口和利用率
```

### 供需匹配法
```
总量 = 已知参照物数量 × 稳定比例系数

例：咖啡豆市场 = 咖啡杯销量 × 15克/杯 × 90克咖啡浆果/15克咖啡豆
例：金融服务市场 = 服务人口 ÷ 77人/从业者
```

## 产能周期判断公式

```
CAPEX = 购建固定资产、无形资产和其他长期资产支付的现金
      - 处置固定资产、无形资产和其他长期资产收回的现金净额

产能周期比值 = CAPEX ÷ (折旧 + 摊销)

判断标准：
> 1.5  → 扩张期，产能持续增加
≤ 1.5  → 格局改善阶段（买入参考）
< 1.0  → 绝对底部，无新增投资（强买入信号）
```

---

## 行业集中度公式

```
CRn = 前n家企业市场份额之和

判断参考：
CR3 > 70%  → 高度集中（寡头格局）
CR3 30-70% → 中度集中
CR3 < 30%  → 分散竞争

对标逻辑：与欧美日韩成熟市场CR值比较，判断国内集中度天花板空间
```

---

## 纵向议价权判断

```
资金占用能力 = (应付账款 + 预收账款 + 合同负债)
             - (应收账款 + 预付账款 + 合同资产)

正值 → 占用上下游资金，议价权强
负值 → 被上下游占用资金，议价权弱
```

---

## 相对估值公式

```
行业相对估值 = 行业市盈率 ÷ 全部A股市盈率

用途：剔除宏观市场整体估值波动，判断行业自身估值水平
```

## 关键阈值汇总

| 指标 | 阈值 | 含义 |
|------|------|------|
| 渗透率 | 15%~20% | 导入期→成长期拐点 |
| 渗透率 | 35%~40% | 成长期→成熟期拐点 |
| CAPEX/折旧摊销 | ≤1.5 | 产能格局改善信号 |
| CAPEX/折旧摊销 | <1.0 | 行业绝对底部 |
| 汽车经销商库存系数 | ≥2.5 | 库存过高，降价风险 |
| 细分TAM | <500亿元 | 不具备孕育独角兽条件 |
| 成长期PE | 80~200倍 | 正常区间，不代表贵 |
| 成熟期PE | 20~40倍 | 正常估值中枢 |
| 交易拥挤度 | >均值+1.5σ | 股价极度危险 |
| 交易拥挤度 | <均值-1.5σ且景气良好 | 买点参考 |
