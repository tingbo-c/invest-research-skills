# 行业典型估值倍数参考

用于行业研究报告的估值判断章节。数字为历史中枢参考，需结合当前市场环境调整。

---

## 使用说明

- 以下为A股/港股历史估值中枢，牛熊市会大幅偏离
- 成长期行业用PS或EV/EBITDA，成熟期行业用PE/股息率
- 周期行业在景气顶部PE最低（盈利最高），在底部PE最高（盈利最低）——不能用PE判断周期股买点

---

## 各行业估值参考

| 行业 | 常用指标 | 历史中枢 | 备注 |
|------|---------|---------|------|
| 消费品（白酒/食品） | PE | 25-40x | 龙头享受溢价 |
| 医药（创新药） | PS / Pipeline价值 | 8-15x PS | 研发管线是核心变量 |
| 医药（仿制药/器械） | PE | 20-30x | — |
| 科技（半导体设计） | PS | 5-15x | 周期波动大 |
| 互联网平台 | PE / EV/EBITDA | 15-25x | 成熟后看现金流 |
| 新能源（光伏/风电） | PE | 15-25x | 强周期，警惕产能过剩 |
| 新能源车 | PS / EV/销量 | — | 渗透率驱动，估值分化大 |
| 银行 | PB | 0.5-1.2x | ROE决定PB中枢 |
| 地产 | PE / NAV折扣 | — | 当前特殊时期，传统方法失效 |
| 工业/制造 | PE / EV/EBITDA | 15-25x | 看ROE稳定性 |
| 公用事业 | PE / 股息率 | 15-20x PE / 4-6% 股息 | 稳定现金流 |
| 传媒/游戏 | PE | 20-35x | 内容周期影响大 |

## 周期行业特别说明

周期行业（钢铁、煤炭、化工、有色、航运）不适合用PE择时：
- 景气顶部：盈利最高 → PE最低 → 看起来"便宜" → 实际是卖点
- 景气底部：盈利最低或亏损 → PE极高或为负 → 看起来"贵" → 实际是买点

**正确方法：** 用PB或EV/EBITDA，结合行业库存周期、价格走势判断周期位置。

## 估值折溢价因素

以下因素会导致个股相对行业中枢溢价或折价：

**溢价因素：**
- 行业龙头地位（市占率持续提升）
- 护城河清晰（品牌/技术/网络效应/转换成本）
- ROE持续高于行业平均
- 管理层诚信记录好

**折价因素：**
- 公司治理问题
- 高度依赖单一客户/产品
- 净利润含金量低（应收账款高、现金流差）
- 行业监管不确定性高
