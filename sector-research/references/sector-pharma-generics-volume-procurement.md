# 仿制药/集采行业分析框架

## 核心分析逻辑

集采把行业从"渠道溢价"改写为**"成本曲线 + 中标份额"的规模生意**。盈利核心变量是降价幅度、供货兑现能力与规则迭代方向。

---

## 行业特有关键指标

| 指标 | 实证数据 | 异常信号 |
|------|---------|---------|
| 国家集采平均降幅 | 前九批平均约 **55%**，多在 50%-60% 区间 | 单品降幅>90% = 必须有极低成本+强放量才能保利润 |
| 中选竞争强度 | 第十一批：55个品种、445家企业、794个产品投标、453个拟中选 | 竞争高度拥挤 = "中标后利润"被价格战压缩 |
| 一致性评价状态 | "视同通过"加速扩大竞标人数 | 同品种竞标人数突增 = 直接改变报价策略与中标概率 |

---

## 景气度领先指标：集采事件链

```
报量公告 → 开标结果 → 各省落地执行文件 → 院内替换执行
```

**关键节点**：地方执行文件可要求"非中选高价药采购量不超过同通用名的 30%"，执行后非中选份额快速出清。盯各省执行进度比盯全国平均降幅更有操作价值。

---

## 估值方法

| 阶段 | 估值方法 | 处理原则 |
|------|---------|---------|
| 开标前 | 情景分析 | 中标 / 不中标 / 复活机制三个情景分别测算 |
| 开标后 | EV/EBITDA 或 FCF 收益率 | 用"供货兑现速度"调整倍数，而非单纯看降价幅度 |

**❌ 不适合用 PE**：集采后盈利波动大，PE 失真严重

---

## 产业链分析重点

关键在**成本底线 + 稳定供货能力**：

- **原料药一体化**：垂直整合度高 → 成本壁垒 → 在55%降幅环境下仍可保持正现金流
- **质量体系**：飞检合规 → 避免中标后被取消资格
- **供货保障**：无法稳定供货 = 丢失份额，且影响后续集采资质

---

## 行业特有分析陷阱

**❌ 把"中标"当"盈利确定"**

集采规则持续迭代（锚点设定、复活机制、质量监管）会改变报价逻辑：
- 历史"唯低价"策略 → 新规"反内卷/稳质量"导向
- 复活机制改变后，历史打法的中标概率模型失效

**正确做法**：每批集采前重新评估规则变化，不能套用历史降幅作为唯一参数。

---

## 政策敏感性

| 政策类型 | 传导路径 |
|---------|---------|
| 集采规则迭代（锚点/复活机制） | 规则 → 报价策略 → 中标格局 → 医院替换 → 利润重估 |
| 一致性评价加速审批 | 竞标人数增加 → 竞争加剧 → 价格进一步下压 |
| 质量监管飞检 | 不合格企业被取消中选资格 → 份额重新分配 |
| 地方执行约束 | 非中选采购量上限 → 出清速度加快 |

**参考来源**：国家医保局第十一批集采通稿；集采质量监管答记者问
