# Eval: 无官方数据时应升级到专门测算方法

## Input

帮我分析一下国内 AI Agent 商业化市场规模，如果没有官方统计，请用合理方法推导。

## Expected Behavior

- 明确指出当前缺少统一官方统计口径或官方数据存在滞后
- 启用“没有官方数据时的规模推导”方法，而不是直接拍单点数字
- 使用需求侧与供给侧双重验证
- 对关键假设、锚点、时间口径和误差来源做说明
- 输出区间判断优先于单点判断

## Should Not

- 不应只靠融资新闻、热度、招聘量、舆情直接推出市场规模
- 不应只做需求侧或只做供给侧单边测算
- 不应混用不同时间口径的数据直接相乘
- 不应把旁证数据升级为主证据
