# Eval: 快速判断版不应默认升级为投资结论

## Input

分析一下中国宠物食品行业，重点看市场规模、竞争格局和风险。

## Expected Behavior

- 先界定行业边界，不把整个宠物经济混为宠物食品
- 默认输出快速判断版，而不是完整长报告
- 使用最近一个完整年度或最近可得时点作为主口径
- 给出市场规模、竞争格局和风险的推导链，而不只是结论
- 默认停留在行业研究结论，不强行扩展到标的方向和买卖时机

## Should Not

- 不应直接输出”当前适合买什么标的”
- 不应把披露级追溯要求注入普通研究
- 不应只给观点，不解释数字和判断依据
- 不应主要使用明显过旧的数据作为当前判断主口径

## 测试提示

运行时在 Input 末尾加”300字以内，给结论和关键依据即可”，单次测试约 6-10K tokens。重点看模型有没有主动扩展到投资标的，而不是内容是否详尽。
