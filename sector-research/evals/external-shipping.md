# Eval: 航运行业应触发外部环境子分析

## Input

（假设用户已回答完启动前问题后）帮我研究中国航运行业，重点看景气度、运价驱动、竞争格局和主要风险。

## Expected Behavior

- 在进入行业结论前，先判断该行业是否需要纳入宏观和外部环境分析
- 明确指出航运属于外部环境高度敏感行业，至少应考虑：
  - 全球贸易量与海外需求
  - 航线扰动、港口限制、绕航风险
  - 运价、运力供给与船期变化
  - 燃油成本、汇率或国际政策约束
- 应加载外部环境相关分析，而不是只停留在国内供需或公司名单
- 输出中应写出传导链：国际局势/全球贸易变化如何影响运价、运力、景气和利润
- 若同时纳入宏观分析，应说明宏观与外部环境各自影响的是哪一段链条

## Should Not

- 不应只做“国内航运公司竞争格局”而跳过国际航线和全球贸易变量
- 不应泛泛而谈“国际局势复杂”，却不说明具体如何传导到行业
- 不应把外部环境分析写成宽泛政治评论，偏离行业研究主题
- 不应漏掉高频验证信号，例如运价指数、吞吐量、船舶供给、绕航情况等

## 测试提示

运行时可在 Input 末尾加一句“先判断是否需要宏观和外部环境子分析，再给结论”。重点验证是否命中 `第二点五步` 和 `第二点六步`，并且外部环境分析不是空泛口号。
