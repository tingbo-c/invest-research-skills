# Eval: 标准报告版应覆盖行业研究主体章节

## Input

请按行业研究报告结构，完整分析中国宠物食品行业，包括行业定义与边界、产业链、市场规模、竞争格局、增长驱动、风险和投资价值。

## Expected Behavior

- 进入标准报告版，而不是快速判断版
- 报告主体至少覆盖：行业定义与边界、产业链结构、市场规模、竞争格局、增长驱动、风险
- 生命周期判断应作为分析前提出现，而不是被完全忽略
- 若用户明确提到“投资价值”，可补投资结论，但应建立在前文行业研究正文之上
- 市场规模应体现主口径、对照口径或交叉校验意识

## Should Not

- 不应退化为几段摘要
- 不应只有结论，没有行业研究正文
- 不应把投资结论写成无前提支撑的“建议关注”
- 不应省略产业链或关键环节分析
