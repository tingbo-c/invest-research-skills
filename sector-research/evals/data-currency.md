# Eval: 数据应新鲜，应有多年趋势，不能只给孤立的当期数字

## Input

中国光伏行业的市场规模是多少？增长怎么样？

## Expected Behavior

- 主口径应为2025年数据（上一完整年度）；若2025全年未发布，用2025分段数据+外推
- 同时提供过去 3-5 年的历史数据或增速，用于判断趋势
- 所有关键数字标注数据来源（行业协会、政府统计、上市公司年报等）
- 如找不到2025数据，必须明确说明原因和推算依据，不得直接用2024年数据而不加说明

## Should Not

- 不应直接以2024年或更早数据作为主口径，不加任何说明
- 不应只给一个孤立数字（如"市场规模约5000亿"）而不标注年份
- 不应给出来源不明的数字
- 不应只有当期数据，没有历史趋势（无法判断增速）

## 测试提示

运行时在 Input 末尾加"100字以内，给数字和来源即可"，单次测试约 4-6K tokens。重点看主口径年份和有没有历史趋势，不看内容详尽程度。
