# Eval: 无专项文件的行业，应先识别核心变量再分析

## Input

（假设用户回答了三个启动问题后）帮我研究中国宠物食品行业的竞争格局

## Context

宠物食品行业没有对应的行业专项 reference 文件。

## Expected Behavior

- 在进入竞争分析之前，先识别并说明该行业的核心变量，包括：
  - 供给侧核心指标（如产能、SKU 数量、品牌数等）
  - 需求侧核心驱动（如养宠人群规模、渗透率、ARPU、细分品类结构等）
  - 景气度领先指标（如电商销量、动销、进口数据等）
  - 利润截留环节（品牌、渠道、原料哪个最强势）
- 变量识别后才进入竞争格局分析
- 竞争分析不应套用通用模板，应结合该行业特性（如品牌信任度、国产替代进度）

## Should Not

- 不应直接套竞争格局通用框架（波特五力 + 玩家列表），跳过行业特性判断
- 不应把宠物食品当制造业或消费品分析，忽略其核心驱动（情感消费、功能升级）
- 不应遗漏宠物食品特有的关键变量（如主粮/零食/保健品分类差异）

## 测试提示

运行时在 Input 末尾加"先列出你认为最重要的5个核心变量，每个一句话"，单次测试约 4-6K tokens。重点看变量识别是否行业特有，而不是通用模板套用。
