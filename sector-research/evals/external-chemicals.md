# Eval: 化工行业应按条件同时判断宏观与外部环境

## Input

（假设用户已回答完启动前问题后）帮我分析中国化工行业，重点看景气周期、成本端变化、盈利弹性和风险。

## Expected Behavior

- 先判断化工行业是否受宏观和外部环境双重驱动，并明确答案
- 至少应覆盖以下变量中的关键项：
  - 油气、煤炭或原材料价格
  - 全球供需缺口与海外装置停复产
  - 出口限制、关税、制裁或国际运输成本
  - 国内信用、地产链、制造业投资或库存周期
- 如果结论依赖原料价格或全球供需，必须纳入外部环境子分析
- 如果结论依赖信用、库存或内需修复，必须纳入宏观子分析
- 输出中应区分：哪些是国内周期因素，哪些是国际因素，二者如何共同传导到价差、开工率、库存和利润

## Should Not

- 不应把化工简单当成纯国内制造业，只做产能和竞争格局
- 不应只提原油价格上涨/下跌，而不解释如何传导到行业盈利
- 不应默认展开完整宏观长报告或国际形势评论，偏离行业结论所需范围
- 不应忽略“不同子行业受驱动不同”这一点，机械套用同一逻辑

## 测试提示

可在 Input 末尾加一句“请显式区分国内宏观驱动与国际外部驱动”。重点验证：该场景会不会同时命中宏观和外部环境判定，但又不过度展开无关内容。
