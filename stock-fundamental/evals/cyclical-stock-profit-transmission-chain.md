# Eval: 周期股必须完成利润传导链分析并给出受益类型判断

## Input

帮我分析一下中国神华的基本面。

## Must Do

- 识别煤价是影响公司利润最核心的外部驱动变量
- 写出完整的利润传导链，至少覆盖：煤价变化 → 售价/成本影响 → 吨毛利变化 → 利润弹性
- 给出明确的受益类型判断，必须是以下五类之一：直接受益 / 间接受益 / 中性 / 反向受损 / 题材映射
- 说明当前该变量方向对公司是顺风还是逆风（或方向不明）
- 给出至少 3 个需要持续跟踪的核心指标（如坑口煤价、港口库存、电厂日耗等）

## Must Not

- 只说"公司业绩受煤炭价格影响较大"，不写出具体传导路径
- 受益类型用模糊表述（如"有一定影响"、"整体偏正面"）代替明确分类
- 跳过传导链，直接跳到财务数据

## Notes

- 不要求对煤价走势做出预测，只要求分析传导机制
- 中国神华是一体化煤电公司，电力业务对煤价波动有部分对冲，分析时应体现这一结构特点
- 重点验证：利润传导链是否写出，受益类型是否明确判断
