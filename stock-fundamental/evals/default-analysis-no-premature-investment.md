# Eval: 普通基本面分析不应默认越权到投资建议

## Input

分析一下海天味业的基本面，我想先知道这家公司商业模式质量怎么样。

## Must Do

- 先做基本面判断，而不是直接给买卖建议
- 识别商业模式类型，并据此组织后续分析重点
- 至少覆盖财务健康度、护城河或持续性、主要风险中的核心项
- 在需要时纳入最低限度的行业背景，而不是孤立看财务指标

## Must Not

- 在用户未明确要求时直接下“买/卖/增持/减持”结论
- 只罗列财务指标，不解释这些指标对商业模式质量意味着什么
- 把估值判断冒充成基本面结论

## Notes

- 允许不同表达顺序，不要求固定章节结构
- 重点验证边界克制和分析完整性，不验证具体观点方向
