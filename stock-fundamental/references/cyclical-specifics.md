# 周期/资源型资产专项分析

本文件仅在公司被判断为资源型或周期性资产时加载。用于补充两项判断：资产类型识别 + 周期位置判断。

## 何时加载

满足以下任一条件时加载：
- 主营产品是大宗商品、能源、基础原材料
- 利润随行业景气大幅波动（峰谷利润差超过 3 倍）
- 核心驱动变量是商品价格或产能周期

## 第一步：识别资产类型

先归类，再确定后续分析重点：

| 类型 | 典型行业 | 分析重点 |
|---|---|---|
| **资源型** | 煤炭、铜、黄金、钾肥、油气、盐湖 | 资源禀赋质量、储量、成本曲线位置 |
| **中游加工型** | 炼化、钢铁深加工、化纤、水泥 | 价差、顺价能力、成本竞争力 |
| **一体化平台型** | 煤化工、磷化工、油炼化一体化 | 各段利润贡献、抗周期底部能力 |
| **强周期制造型** | 工程机械、重卡、船舶、部分设备 | 开工率、订单、固定成本摊薄弹性 |
| **周期成长混合型** | 资源+新材料、化工+半导体材料 | 成长逻辑兑现进度 + 周期位置双判断 |

## 第二步：判断当前周期位置

从以下五个阶段中选一个，并说明判断依据：

| 阶段 | 特征信号 | 分析关注点 |
|---|---|---|
| **底部** | 价格低位、利润微薄或亏损、市场预期差 | 供给是否开始收缩？库存是否去化？成本支撑在哪里？ |
| **复苏初期** | 价格开始反弹、开工率回升、价差改善 | 弹性最大阶段，谁先受益？需求修复可持续吗？ |
| **景气上行** | 量价齐升、利润快速增长、龙头被追捧 | 供需缺口还能持续多久？新增供给何时出现？ |
| **高位繁荣** | 利润很高、资本开支增多、扩产加速 | 价格是否已透支？政策是否会干预？需求是否见顶？ |
| **下行期** | 价格回落、利润收缩、估值业绩双杀 | 哪些公司有底仓价值？谁先出清？谁能穿越周期？ |

**判断时注意：**
- 不同品种可能处于不同周期阶段，不要用单一行业周期代替公司判断
- 周期成长混合型公司，成长逻辑可能覆盖部分周期下行风险，要分开评估

## 第三步：评估公司在周期中的相对位置

回答：

1. **成本曲线位置**：公司是行业低成本方（穿越周期）还是边际成本方（高位时利润好，底部时最先出清）？
2. **弹性来源**：在当前周期阶段，利润弹性主要来自价格、价差、产量还是成本摊薄？
3. **底部韧性**：若周期下行，公司能否维持正现金流或基本盈亏平衡？

## 输出要求

加载本文件后，至少补充：
1. 公司属于五类中的哪一类（资产类型）
2. 当前处于周期五阶段中的哪一段，判断依据是什么
3. 公司在成本曲线上的位置（低成本/中段/边际成本方）

## 一句话原则

同一家公司，在周期底部和高位，买卖逻辑完全不同。先判断周期位置，再判断公司质量，否则再好的公司也可能买在错误的时间。
