# 利润传导链与受益类型判断

本文件用于第四步：识别外部驱动变量，分析其如何传导到利润表，并判断受益类型。

## 使用前提

只有当公司利润受外部变量明显影响时才需要完整走一遍。判断标准：
- 产品定价与某类商品价格/利率/汇率强相关
- 原材料成本在总成本中占比超过 30%
- 需求明显受政策周期、宏观周期驱动
- 公司近 3 年利润波动与某类外部变量走势基本同步

若公司利润主要由内部运营改善（份额提升、产品升级、成本管控）驱动，此文件简答即可。

## 第一步：识别外部驱动变量

找出对这家公司利润影响最大的 1-3 个变量，从以下类别中选取最相关的：

| 变量类别 | 常见例子 |
|---|---|
| 商品/能源价格 | 煤价、铜价、油价、天然气、钾肥、氨 |
| 原材料成本 | 石脑油、废钢、纯碱、钛白粉、电价 |
| 利率/信用环境 | 基准利率、信贷收紧、再融资成本 |
| 汇率 | 人民币兑美元、出口结算货币 |
| 政策变量 | 医保谈判、环保限产、补贴退坡、关税 |
| 需求/宏观周期 | 地产投资、基建开工、消费信心指数 |
| 技术替代 | 新技术渗透率、老技术淘汰速度 |

## 第二步：写出利润传导链

对每个识别出的外部变量，写出完整传导路径：

```
外部变量变化
    ↓
影响哪个环节（产品售价 / 原料成本 / 需求量 / 产能利用率）
    ↓
单位盈利变化（吨毛利 / 价差 / 加工费 / 净息差）
    ↓
板块或公司整体利润变化（放大 / 压缩 / 中性）
```

**示例（资源型公司）：**
铜价上涨 → 销售均价提升 → 单吨毛利扩大（成本短期固定）→ 利润高弹性释放

**示例（加工型公司）：**
原料价格上涨 → 成本端承压 → 若顺价滞后，价差收窄 → 吨毛利下降 → 利润承压

**示例（消费品公司）：**
原材料涨价 → 毛利率压缩 → 若品牌力强可提价转嫁 → 净利率基本稳定；若提价受阻 → 利润下滑

## 第三步：判断受益类型

完成传导链后，必须给出明确分类，不能模糊带过：

| 类型 | 判断标准 |
|---|---|
| **直接受益** | 外部变量变化直接映射到利润放大，传导路径短且强 |
| **间接受益** | 通过成本降低、需求改善或竞争格局优化获益，传导需要 1-2 个中间环节 |
| **中性** | 有受益端也有受损端，大致抵消；或公司利润对该变量不敏感 |
| **反向受损** | 外部变量变化导致成本上升或收入下降，且无法有效转嫁 |
| **题材映射** | 概念/行业上相关，但利润表实际敏感度弱，炒的是预期而非基本面 |

**最容易犯的错：把"题材映射"判断成"直接受益"。**
验证方法：找近 3 年该变量大幅波动的时期，看公司利润是否同向大幅变动。若波动不明显，大概率是题材映射。

## 输出要求

完成本文件分析后，至少写清楚：
1. 核心外部驱动变量是什么（1-3 个）
2. 每个变量的传导链（一句话描述路径）
3. 受益类型判断（五选一，不能写"部分受益"之类模糊表述）
4. 当前该变量处于什么方向（有利 / 不利 / 方向不明）

## 一句话原则

不问这家公司有没有蹭上热点，要问这个变量变化会不会让这家公司的吨毛利/价差/净息差真实扩大。
